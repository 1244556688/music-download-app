========================================
🚀 打包完全教學指南：
========================================

第一步：準備必備環境與檔案
1. 請先確保你的電腦有安裝 Python 環境。
2. 打開 CMD 視窗，安裝打包與執行必備的套件：
   pip install flask yt-dlp pyinstaller

第二步：放入 FFmpeg (最關鍵)
1. 請將你桌面或電腦裡的「ffmpeg.exe」複製一份。
2. 貼到本資料夾內，跟 `app.py` 放在同一個目錄底下。
   (此時本目錄應該要有：app.py, config.py, utils.py, ffmpeg.exe ... 等)

第三步：一鍵完美打包指令
1. 打開 CMD 視窗，並切換路徑到這個解壓後的資料夾：
   cd /d "你解壓這個專案的完整資料夾路徑"

2. 直接複製並執行下方這行「終極打包指令」：
   pyinstaller --onefile --noconsole --clean --add-data "templates;templates" --add-data "ffmpeg.exe;." app.py

第四步：大功告成！
1. 打包結束後，本資料夾下會出現一個名為 `dist` 的資料夾。
2. 點進去 `dist`，裡面的 `app.exe` 就是你的終極完全體！
