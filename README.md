# ⚡ 專業級影音下載器 V3 - Web All-in-One 完全體

這是一款基於 **Flask** 與 **yt-dlp** 核心開發的專業影音下載器。本版本特別針對「無程式背景之一般使用者」進行了極致的優化，擁有精美的網頁控制台介面，並支援將所有依賴套件（包含解碼引擎 FFmpeg）完美融合成單一一個獨立的 `.exe` 執行檔！

## ✨ 專案特色

- 🖥️ **精美網頁控制台**：免去傳統 CMD 複雜指令，全網頁視覺化操作，支援即時下載進度條、速度與剩餘時間顯示。
- 📦 **單一獨立 EXE 化**：支援將 Python 邏輯、Flask 靜態網頁（HTML/JS）以及 `ffmpeg.exe` 壓縮包裝成單一執行檔，達成「免安裝環境、隨點隨用」。
- 📁 **動態路徑客製化**：網頁端支援即時輸入並更新儲存路徑，防呆防錯。
- 🎚️ **頂級規格自由配**：支援最高 8K 解析度、60fps 幀率下載，並提供主流影片編碼（H.264, H.265, AV1, VP9）與多種音訊格式（MP3, FLAC, WAV）的無損/高位元率轉換。
- 🤫 **無 console 背景執行**：打包後可達成完全隱藏黑視窗，直接自動彈出網頁，體驗如同原生 Windows 軟體。
